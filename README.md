# Сравнение моделей компьютерного зрения для детектирования объектов

## Описание проекта

Проект выполнен в рамках учебной практики по направлению "Искусственный интеллект и машинное обучение". Цель работы — реализовать полный pipeline решения задачи компьютерного зрения и сравнить современные модели детектирования объектов.

## Задача

Рассматривается задача детектирования объектов на изображениях. На вход модели подается изображение, на выходе формируются координаты ограничивающих рамок, классы объектов и значения уверенности модели.

## Используемый набор данных

В работе используется набор данных COCO128, представляющий уменьшенную версию датасета COCO. Датасет содержит изображения с аннотациями объектов и применяется для быстрого обучения и тестирования моделей детектирования.

## Используемые модели

- YOLOv8n
- YOLOv8s
- YOLOv8m
- YOLOv9t / YOLOv10n / YOLO11n
- YOLO11s

## Метрики качества

- Precision
- Recall
- F1-score
- mAP50
- mAP50-95

## Структура проекта

- `configs/` — конфигурационные файлы
- `src/dataset/` — работа с данными
- `src/models/` — описание моделей
- `src/training/` — обучение моделей
- `src/evaluation/` — оценка качества
- `results/` — результаты экспериментов
- `notebooks/` — экспериментальные ноутбуки
- `report/` — отчет по практике
- `presentation/` — презентация
